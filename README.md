# Sera Landing Page

Landing page for Sera - the photobooth app that brings back the magic of capturing imperfect moments.

## Production Checklist

### Before Deployment

- [ ] Update domain in all meta tags (search for `sera.camera`)
- [ ] Replace App Store link in index.html
- [ ] Update social media links (TikTok, Instagram, X)
- [ ] Test all links and forms
- [ ] Optimize all images (already using SVGs)
- [ ] Test on mobile devices
- [ ] Test in multiple browsers (Safari, Chrome, Firefox)
- [ ] Set up SSL certificate (HTTPS)

### SEO Setup

- [ ] Submit sitemap.xml to Google Search Console
- [ ] Verify domain ownership in Search Console
- [ ] Set up Google Analytics (optional)
- [ ] Test structured data with Google Rich Results Test
- [ ] Verify Open Graph tags with Facebook Debugger
- [ ] Verify Twitter Cards with Twitter Card Validator

### Performance

- [ ] Test with Google PageSpeed Insights
- [ ] Verify Core Web Vitals
- [ ] Enable CDN (optional)
- [ ] Enable Gzip compression (included in .htaccess)

### Monitoring

- [ ] Set up uptime monitoring
- [ ] Monitor Search Console for errors
- [ ] Track organic traffic

## Files

- `index.html` - Main landing page
- `privacy.html` - Privacy policy page
- `style.css` - Styles for both pages
- `script.js` - Animations and interactions
- `robots.txt` - Search engine crawling rules
- `sitemap.xml` - Sitemap for search engines
- `.htaccess` - Server configuration (Apache)
